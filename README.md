# revo
Mario Kart Wii Discord Rich Presence

``Using the decompilation of Mario Kart Wii, this is my Discord Rich Presence extension for Mario Kart Wii``

``Created using my own DLL calling conventions to read and write memory dynamically. This allows me to make further alterations to the way in which the memory is parsed through the program.``
